# HTML Tools Collection (HTML 工具箱)

这是一个自用的单页面 Web 工具集合。

本项目深受 [Simon Willison's HTML Tools](https://simonwillison.net/2025/Dec/10/html-tools/) 的启发。核心理念是：**每一个工具都是一个独立的 HTML 文件**。

## 核心哲学

- **单文件交付 (Single File)**: HTML、CSS 和 JavaScript 全部包含在一个文件中，没有复杂的构建步骤，下载即用。
- **无后端 (No Backend)**: 所有逻辑完全在浏览器端运行，确保隐私安全，无需服务器支持。
- **AI 辅助构建 (AI Generated)**: 利用 LLM (如 ChatGPT, Claude, Gemini) 快速实现功能原型并迭代完善。

## 如何使用

每个工具都是独立的 `.html` 文件。
1. 直接点击文件名在 GitHub 中查看（或通过 GitHub Pages 访问）。
2. 或者下载到本地，双击在浏览器中打开即可运行。
