HTML 小工具集

一个自用的单页面 Web 工具集合，灵感来源于 Simon Willison's HTML Tools。 每个工具都是独立的 HTML 文件，完全在浏览器端运行，无后端依赖，并主要通过 AI 辅助构建。

工具列表

  • HTML Preview 一个预览 HTML 的工具
  • HTML Simplifier 简化 HTML，删除无用信息，减少 Token 消耗
  • Code Prettier 代码格式化工具 (支持 JSON, JS, CSS 等)
  • Image to WebP 图片转 WebP 工具 (支持 JPG/PNG)
  • Markdown Preview Markdown 实时预览工具
  • Viewport Size 查看设备 Viewport 尺寸和屏幕信息
  • Mobile Viewer 选择常见手机设备并预览指定 URL
  • QR Reader QR 码扫描工具 (支持摄像头与图片上传)
